#  E-Commerce Sales Analysis | MySQL + Power BI

##  Project Overview

This project analyzes the Sample Superstore dataset using MySQL and Power BI to uncover business insights related to sales, profit, customers, products, and regional performance. SQL was used for data analysis, while Power BI was used to build an interactive dashboard for effective business reporting.


##  Tools & Technologies

- MySQL
- Power BI
- Microsoft Excel


##  Dataset

- Dataset: Sample Superstore
- Total Records: 9,994
- Total Columns: 21


##  SQL Analysis

The following business questions were answered using SQL:

- Total Records
- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Sales by Category
- Top 10 Products by Sales
- Top 10 Customers by Sales
- Most Profitable Categories
- Loss-Making Products
- Sales by Region
- Profit by State
- Top 10 Cities by Sales
- Sales by Customer Segment
- Average Discount by Category
- Impact of Discount on Profit

### SQL Concepts Used

- SELECT
- GROUP BY
- ORDER BY
- HAVING
- Aggregate Functions (SUM, COUNT, AVG)
- DISTINCT
- LIMIT
- ROUND


##  Power BI Dashboard

The dashboard includes:

- KPI Cards (Total Sales, Total Profit, Total Orders, Total Customers, Profit Margin)
- Monthly Sales Trend
- Sales by Category
- Profit by Category
- Sales by Region
- Top Selling Products
- Sales by Customer Segment


##  Key Insights

- Technology category generated the highest sales.
- The West region contributed the highest revenue.
- A small number of products generated a significant portion of total sales.
- Higher discounts generally resulted in lower profitability.
- Customer purchasing behavior varied across different segments.
